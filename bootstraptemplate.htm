Bootstrap Templates

I searched for Free BootStrap Templates on google and found this website Start BootStrap. The great thing about the templates they offer is that they can have you view the template live and can also view source to see the index.html file and the css that goes into making the template itself. I think viewing the source files are helpful, when I looked at the files for this template you can see the four areas that I would change for the topics I used in making my menu last time we worked on project 0. The template I would use is called Heroic Features.Heroic Features It has four cards with a little description of each of the cards about what is in them and then it gives a button that says "find out more!" which I assume would be to redirect you to viewing more of the information. I wouldn't use all of the templates features but I know for a fact I would use the different cards.
